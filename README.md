# Malaysia House Price Prediction

An end-to-end machine learning project for predicting Malaysia house prices using Python, Pandas, visualization, and Scikit-learn.

## Project Overview

This project demonstrates the workflow of machine learning prediction, including:

* Data preprocessing
* Data cleaning
* Feature engineering
* Exploratory Data Analysis (EDA)
* Model training and evaluation
* Interactive house price prediction

The prediction results are intended for educational and reference purposes only, as the dataset may contain incomplete or limited information.

## Dataset

Dataset source from Kaggle:

https://www.kaggle.com/datasets/lyhatt/house-prices-in-malaysia-2025

Credits to the original dataset creator for providing the public dataset.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Jupyter Notebook

## How to Run

1. Download the dataset from the Kaggle link above.
2. Place the CSV file in the same directory as the Jupyter Notebook file.
3. Open the notebook in Jupyter Notebook.
4. Run all cells sequentially.

Example:

```python
df = pd.read_csv('malaysia_house_price_data_2025.csv')
```
